# Curso Introdutório de Python para Ciência de Dados

## Sobre o projeto e objetivo

Este repositório foi criado para armazenar e organizar os materiais de um curso introdutório de Python com foco em Ciência de Dados. A proposta do curso é apresentar, de forma simples e progressiva, os fundamentos da programação em Python e sua aplicação inicial em atividades relacionadas à análise de dados.

O material foi estruturado para servir de apoio ao desenvolvimento das aulas, à prática dos participantes e à organização do conteúdo produzido pela equipe.

## Escopo do curso

O curso foi definido com foco em participantes iniciantes, sem exigir experiência prévia com programação. A proposta é construir uma base sólida nos fundamentos de Python e avançar, de forma gradual, até o contato inicial com análise exploratória de dados.

O conteúdo foi delimitado para manter o caráter introdutório do projeto. Assim, o curso aborda conceitos básicos de programação, manipulação inicial de dados, visualização de informações e exploração introdutória de conjuntos de dados.

Não fazem parte do escopo temas mais avançados, como aprendizado de máquina, desenvolvimento web, banco de dados ou assuntos voltados a níveis mais avançados da área.

## Ementa do curso

### Módulo 1 – Fundamentos de Python
Introdução à linguagem, sintaxe básica, variáveis, tipos de dados, operadores, entrada e saída, estruturas condicionais, estruturas de repetição, listas, tuplas, dicionários e noções iniciais de organização de código.

### Módulo 2 – Manipulação de Dados
Leitura de arquivos, organização de dados em tabelas, seleção de colunas e linhas, filtros simples, criação de colunas e operações iniciais de tratamento de dados.

### Módulo 3 – Visualização de Dados
Construção e interpretação de gráficos simples, com foco na apresentação visual de informações e no apoio à compreensão dos dados.

### Módulo 4 – Introdução à Análise Exploratória de Dados
Observação de padrões, identificação de comportamentos básicos em conjuntos de dados, interpretação inicial de resultados e formulação de percepções a partir das informações analisadas.

## Pré-requisitos

O curso foi planejado para iniciantes. Por isso, não é necessário ter conhecimento prévio em programação.

Espera-se apenas que o participante:

- saiba utilizar computador de forma básica;
- consiga acessar navegador e internet;
- tenha familiaridade mínima com arquivos e pastas;
- possa executar os materiais em ambiente apropriado, como Google Colab ou Jupyter Notebook.
