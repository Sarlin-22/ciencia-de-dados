# Curso Python para Ciencia de Dados

Repositorio com materiais do curso introdutorio de Python para Ciencia de Dados. O conteudo esta organizado por modulos, com notebooks, exercicios, datasets e um projeto final de analise exploratoria.

## Descricao do curso

O curso apresenta uma trilha pratica para aprender Python aplicado a dados:

- fundamentos de Python;
- manipulacao de dados com pandas;
- visualizacao de dados com matplotlib e seaborn;
- analise exploratoria de dados;
- formulacao de perguntas de negocio;
- comunicacao de insights em notebooks.

## Como executar

1. Abra este repositorio no VS Code.
2. Crie e ative um ambiente virtual, se desejar:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Instale as bibliotecas usadas no curso:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

4. Abra o notebook desejado no VS Code ou Jupyter Notebook.
5. Execute as celulas em ordem.
